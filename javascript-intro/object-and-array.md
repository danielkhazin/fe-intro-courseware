# Object & Array

The object is a data type that allows you to store collections of data, e.g.:

```
let task = {
    taskName: 'My Name',
    date: '2020-12-01',
    status: 'Pending',
    mark: '0%'
};
```

An array is a type of object used for storing multiple values in single variable, e.g.:

```
let tasks = [{
    taskName: 'Math Test',
    date: '2020-12-01',
    status: 'Pending',
    mark: '0%'
  },
  {
    taskName: 'Science Test',
    date: '2020-12-01',
    status: 'Finished',
    mark: '100%'  
  }
];
```
